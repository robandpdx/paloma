================================================================================
              BUTTON POSITION SWAP - IMPLEMENTATION SUMMARY
================================================================================

ISSUE REQUEST:
"Move the Reset button that is in the Repository Settings modal to where the 
Delete button is now. Move the Delete button into the Repository Settings 
modal, to where the Reset button is now."

STATUS: ✅ COMPLETE

================================================================================
                          WHAT WAS CHANGED
================================================================================

1. RESET BUTTON
   Location Change:
   - FROM: Repository Settings modal footer
   - TO:   Main page repository actions (middle button)
   
   Behavior:
   - Click opens ResetConfirmationModal
   - User must confirm by clicking "Reset" in modal
   - Modal shows detailed list of what will be reset
   - Disabled when state is 'pending' or 'reset'
   - Includes tooltip explaining why disabled
   
   Code Changes:
   - Added resetRepo state to track repository for confirmation
   - Reset button calls setResetRepo(repo) on click
   - ResetConfirmationModal renders when resetRepo is not null
   - On confirm: calls resetRepository(resetRepo) and closes modal

2. DELETE BUTTON
   Location Change:
   - FROM: Main page repository actions
   - TO:   Repository Settings modal footer
   
   Behavior:
   - Click opens DeleteModal (from within SettingsModal)
   - User must type repository URL to confirm
   - Always enabled (can delete any repository)
   - Extra safety: must open Settings modal first
   
   Code Changes:
   - Removed deleteRepo state (no longer needed)
   - SettingsModal now receives onDelete prop instead of onReset
   - Delete button in SettingsModal footer calls onDelete
   - DeleteModal shown from within SettingsModal when clicked

                         USER EXPERIENCE

WORKFLOW CHANGES:

BEFORE: Click Settings → Click Reset → Confirm in modal

Delete Repository:
BEFORE: Click Delete → Type URL in modal
AFTER:  Click Settings → Click Delete → Type URL in modal
RESULT: One extra step, more protection
